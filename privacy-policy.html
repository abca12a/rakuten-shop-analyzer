Privacy Policy for Rakuten Shop Analyzer

Effective Date: December 2024
Last Updated: December 2024

1. Introduction

Rakuten Shop Analyzer ("we," "our," or "the Extension") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, store, and protect your information when you use our Chrome extension for extracting and analyzing Rakuten shop data.

2. Information We Collect

2.1 Authentication Information

  • Rakuten Application ID: Your API credentials for accessing Rakuten's public web services
  • Purpose: Required to authenticate API requests and enable data extraction functionality
  • Storage: Stored locally in your browser using Chrome's secure storage API

2.2 Website Content

  • Product Information: Names, prices, descriptions, categories, tags, and specifications from Rakuten shops
  • Product Images: High-resolution image URLs and metadata
  • Shop Data: Shop information and product rankings
  • Purpose: To organize, analyze, and export data for business analysis purposes

2.3 Configuration Data

  • User Preferences: Extension settings, processing modes, and timeout values
  • Task History: Status and progress information for completed operations

3. How We Use Your Information

3.1 Core Functionality

  • Authenticate requests to Rakuten's public API using your provided credentials
  • Extract and organize product data from Rakuten shop pages
  • Process and export data in CSV/JSON formats for your analysis
  • Provide real-time progress updates and task notifications

3.2 User Experience

  • Remember your preferences and settings
  • Display appropriate guidance and error messages
  • Maintain task history for reference

4. Data Storage and Security

4.1 Local Storage Only

  • No Remote Servers: All data is stored locally in your browser
  • No Data Transmission: We do not transmit your data to any external servers (except Rakuten's official API)
  • User Control: You have complete control over your data and can delete it at any time

4.2 Security Measures

  • Data is stored using Chrome's secure storage mechanisms
  • API credentials are encrypted and protected by browser security
  • No sensitive information is logged or transmitted to third parties
  • All processing occurs locally within your browser environment

5. Data Sharing and Third Parties

5.1 No Data Sharing

  • We do not sell, rent, or share your personal information with third parties
  • We do not use analytics, tracking, or telemetry services
  • No user identification or profiling is performed

5.2 Rakuten API Interaction

  • Your API credentials are used only to authenticate with Rakuten's official API endpoints
  • API requests are made directly from your browser to Rakuten's servers
  • We do not intercept, store, or process API responses beyond local functionality
  • Subject to Rakuten's own privacy policy and terms of service

6. Data Retention and Deletion

6.1 User Control

  • Manual Deletion: Clear specific shop data or all stored data through the extension interface
  • Automatic Removal: All data is automatically deleted when you uninstall the extension
  • No Backup: No automatic backup or synchronization to external services

6.2 Data Lifecycle

  • Configuration data: Retained until manually deleted or extension uninstalled
  • Extracted product data: Retained until manually cleared by user
  • Temporary processing data: Automatically cleaned after operations complete

7. Permissions Explained

7.1 Required Permissions

  • Storage: Save configuration and extracted data locally
  • Downloads: Export CSV/JSON files to your computer
  • Tabs: Open product pages for image extraction during batch processing
  • Notifications: Inform you when tasks complete or encounter errors
  • Alarms: Schedule cleanup tasks and manage processing timeouts

7.2 Host Permissions

  • Rakuten Domains: Access *.rakuten.co.jp and *.rakuten.com to extract product information
  • Limited Scope: Permissions restricted to Rakuten domains only
  • Purpose: Essential for core data extraction functionality

8. Children's Privacy

This extension is not intended for use by children under 13 years of age. We do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us.

9. Changes to This Privacy Policy

We may update this privacy policy from time to time to reflect changes in our practices or applicable laws. We will notify users of any material changes through extension updates or other appropriate means. The updated version will be indicated by an updated "Last Updated" date at the top of this policy.

10. Compliance

  • This extension fully complies with Chrome Web Store Developer Program Policies
  • Regular reviews ensure continued compliance with evolving requirements

10.2 Data Protection

  • Follows industry best practices for data protection and user privacy
  • Implements privacy-by-design principles
  • Regular security assessments and updates
